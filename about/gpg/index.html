    GPG Signature

    All releases are signed with the GPG key below. All releases include a detached key as well to verify your download against.

    Releases also include a SHA256 Checksum file which is also GPG signed and includes a detached key file to verify the integrity of your download.

    This public key can also be used to encrypt confidential issue reports such as vulnerability reports. See the security policy for details.
